=============================
GitHubの準備(アカウント作成)
=============================

gitを使う際に、情報を取得・提出するための領域(リポジトリ)へのアクセスが必要です。
gitを扱う組織では、組織内のリポジトリを使うことになりますが、ここではオープンなリポジトリサイトであるGitHubを使用します。
そのため、GitHubのアカウントを作成し、設定する必要があります。
